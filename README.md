# Security Hub — Web App

A multi-tool security web application built with vanilla HTML, CSS, and JavaScript. Developed as an integrative project during the SoyHenry Full Stack bootcamp prep course.

## Features

- 🔐 **Password Generator** — Configurable length (3–10 chars), supports uppercase, numbers, and special characters
- 🗄️ **Safe Vault** — Secure local storage for credentials and sensitive notes
- 📊 **Activity Monitor** — Tracks and logs user activity sessions

## Live Demo

Open `index.html` directly in your browser — no build step required.

## Tech Stack

- HTML5
- CSS3 (custom styling, no frameworks)
- Vanilla JavaScript (ES6+)
